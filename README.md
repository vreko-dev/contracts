# ⚠️ Repository Archived

This package has been superseded by the [workspace.json](https://workspacejson.dev) open standard.

## Migration Information

The protocol shapes and type definitions previously defined in this repository are now maintained as part of the workspace.json specification at:

- **Specification**: [workspacejson.dev](https://workspacejson.dev)
- **NPM Package**: [@workspacejson/spec](https://www.npmjs.com/package/@workspacejson/spec)

## What This Means

- This repository is kept for historical reference
- No further updates will be made to this codebase
- All new development happens in the workspace.json ecosystem
- The npm package `@snapback-oss/contracts` has been deprecated

## Next Steps

If you were using this package, please migrate to the workspace.json standard:

1. Visit [workspacejson.dev](https://workspacejson.dev) for the new specification
2. Install `@workspacejson/spec` for type definitions
3. Update your code to use the new workspace.json protocol shapes
